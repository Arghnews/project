6.6. Host Write Ordering Guarantees

When submitting batches of command buffers to a queue via vkQueueSubmit, it is guaranteed that:

  • Host writes to mappable device memory that occurred before the call to vkQueueSubmit are visible to the queue operation resulting from that submission, if the device memory is coherent or if the memory range was flushed with vkFlushMappedMemoryRanges.
